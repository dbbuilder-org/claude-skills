# HECVAT Assessment Skill

Perform a complete HECVAT (Higher Education Community Vendor Assessment Toolkit) analysis of the current codebase and produce two artifacts:

1. `hecvat/HECVAT-YYYY-MM-DD.md` — structured Markdown report with per-section control tables
2. `hecvat/HECVAT-YYYY-MM-DD.xlsx` — Excel workbook generated by a Python script

Determine today's date at runtime using `date +%Y-%m-%d` and use it as YYYY-MM-DD throughout.

If the user said "hecvat lite" or "lite", run only sections 2, 3, 4, 5, 6, and 10. Otherwise run all 13 sections.

## Phase 1: Discovery

Before assessing any control, gather evidence from the codebase. Run all of the following discovery steps. Record findings as you go — do not rely on memory.

Run these commands to orient yourself:


## Phase 2: Evidence Evaluation

After completing all discovery steps, evaluate each HECVAT control. Use only evidence found in Phase 1. Never fabricate compliance certifications or security capabilities that are not evidenced in the code.

Rating scale:
- **PASS** — control is implemented and evidence is clear
- **PARTIAL** — control is partially implemented or evidence is ambiguous
- **FAIL** — control is not implemented or explicitly absent
- **N/A** — control does not apply to this product or deployment model

The script structure:

```python
#!/usr/bin/env python3
"""
HECVAT Excel Generator
Reads hecvat/HECVAT-YYYY-MM-DD.md and produces hecvat/HECVAT-YYYY-MM-DD.xlsx
Run: python3 hecvat/generate-excel.py
"""

import subprocess
import sys
import os
import glob
from datetime import date

# Install openpyxl if needed
try:
    import openpyxl
except ImportError:
    subprocess.check_call([sys.executable, "-m", "pip", "install", "openpyxl"])
    import openpyxl

from openpyxl import Workbook
from openpyxl.styles import (
    PatternFill, Font, Alignment, Border, Side
)
from openpyxl.utils import get_column_letter

TODAY = date.today().strftime("%Y-%m-%d")
SCRIPT_DIR = os.path.dirname(os.path.abspath(__file__))
MD_PATH = os.path.join(SCRIPT_DIR, f"HECVAT-{TODAY}.md")
XLSX_PATH = os.path.join(SCRIPT_DIR, f"HECVAT-{TODAY}.xlsx")

# Color fills
FILL_PASS    = PatternFill("solid", fgColor="C6EFCE")   # green
FILL_PARTIAL = PatternFill("solid", fgColor="FFEB9C")   # yellow
FILL_FAIL    = PatternFill("solid", fgColor="FFC7CE")   # red
FILL_NA      = PatternFill("solid", fgColor="D9D9D9")   # gray
FILL_HEADER  = PatternFill("solid", fgColor="4472C4")   # blue header
FILL_SECTION = PatternFill("solid", fgColor="BDD7EE")   # light blue section header

FONT_HEADER  = Font(bold=True, color="FFFFFF", size=11)
FONT_SECTION = Font(bold=True, size=11)
FONT_NORMAL  = Font(size=10)

THIN_BORDER = Border(
    left=Side(style="thin"),
    right=Side(style="thin"),
    top=Side(style="thin"),
    bottom=Side(style="thin"),
)

def rating_fill(rating_text):
    r = rating_text.strip().upper()
    if "PASS" in r:
        return FILL_PASS
    elif "PARTIAL" in r or "WARN" in r:
        return FILL_PARTIAL
    elif "FAIL" in r:
        return FILL_FAIL
    elif "N/A" in r or "NA" in r:
        return FILL_NA
    return None

def set_col_widths(ws, widths):
    for i, w in enumerate(widths, 1):
        ws.column_dimensions[get_column_letter(i)].width = w

def add_header_row(ws, headers, row=1):
    for col, h in enumerate(headers, 1):
        cell = ws.cell(row=row, column=col, value=h)
        cell.fill = FILL_HEADER
        cell.font = FONT_HEADER
        cell.alignment = Alignment(wrap_text=True, vertical="center", horizontal="center")
        cell.border = THIN_BORDER
    ws.row_dimensions[row].height = 22

def parse_md_tables(md_path):
    """
    Parse the Markdown file and extract section tables.
    Returns: dict mapping section_title -> list of row dicts
    """
    sections = {}
    current_section = None
    headers = []
    rows = []
    in_table = False

    with open(md_path, "r", encoding="utf-8") as f:
        lines = f.readlines()

    for line in lines:
        stripped = line.strip()

        # Detect section heading
        if stripped.startswith("## Section ") or stripped.startswith("## Executive Summary") or stripped.startswith("## Section-Level Roll-Up") or stripped.startswith("## Gap Analysis"):
            if current_section and rows:
                sections[current_section] = {"headers": headers, "rows": rows}
            current_section = stripped.lstrip("# ").strip()
            headers = []
            rows = []
            in_table = False
            continue

        # Detect table header row
        if stripped.startswith("|") and "---" not in stripped and not in_table:
            parts = [p.strip() for p in stripped.split("|") if p.strip()]
            if parts:
                headers = parts
                in_table = True
            continue

        # Skip separator row
        if stripped.startswith("|") and "---" in stripped:
            continue

        # Data row
        if stripped.startswith("|") and in_table:
            parts = [p.strip() for p in stripped.split("|") if p.strip()]
            if parts and len(parts) >= 2:
                row = {}
                for i, h in enumerate(headers):
                    row[h] = parts[i] if i < len(parts) else ""
                rows.append(row)
            continue

        # Non-table line breaks the table
        if in_table and not stripped.startswith("|"):
            in_table = False

    # Capture last section
    if current_section and rows:
        sections[current_section] = {"headers": headers, "rows": rows}

    return sections

def write_summary_sheet(wb, sections):
    ws = wb.active

def write_section_sheet(wb, section_title, section_data):
    # Sheet name: truncate to 31 chars (Excel limit)
    sheet_name = section_title[:31]
    ws = wb.create_sheet(title=sheet_name)

    # Section title row
    ncols = len(section_data["headers"])
    ws.merge_cells(start_row=1, start_column=1, end_row=1, end_column=max(ncols, 1))
    title_cell = ws.cell(row=1, column=1, value=section_title)
    title_cell.fill = FILL_SECTION
    title_cell.font = FONT_SECTION
    title_cell.alignment = Alignment(horizontal="left", vertical="center")
    ws.row_dimensions[1].height = 22

    # Headers
    add_header_row(ws, section_data["headers"], row=2)

    # Data rows
    for r_idx, row in enumerate(section_data["rows"], 3):
        for c_idx, header in enumerate(section_data["headers"], 1):
            val = row.get(header, "")
            cell = ws.cell(row=r_idx, column=c_idx, value=val)
            cell.font = FONT_NORMAL
            cell.alignment = Alignment(wrap_text=True, vertical="top")
            cell.border = THIN_BORDER
            # Color rating column (look for "Rating" header)
            if header.strip().lower() == "rating":
                fill = rating_fill(val)
                if fill:
                    cell.fill = fill
        ws.row_dimensions[r_idx].height = 36

    # Auto-width: sample values for each column
    for c_idx, header in enumerate(section_data["headers"], 1):
        max_len = len(header)
        for row in section_data["rows"]:
            val = row.get(header, "")
            max_len = max(max_len, min(len(val), 60))
        ws.column_dimensions[get_column_letter(c_idx)].width = min(max_len + 4, 65)

    ws.freeze_panes = "A3"
```

## Rules

1. Never fabricate compliance certifications, penetration test results, or security controls not evidenced in the codebase or explicitly stated by the user.
2. When evidence is absent or ambiguous, rate PARTIAL or FAIL with a specific gap note. Do not default to PASS.
3. Every rating must cite a specific file path, code pattern, configuration value, or documentation reference. "No evidence found" is a valid evidence entry.
4. For controls that are inherently operational (physical security, IRP, certifications), rate N/A or PARTIAL and note "Requires organizational verification — not determinable from codebase alone."
5. HECVAT Lite covers only Sections 2, 3, 4, 5, 6, and 10. If the user said "lite" or "hecvat lite", skip Sections 1, 7, 8, 9, 11, 12, and 13. Update the Section Roll-Up table accordingly.
6. Save all output to the `hecvat/` directory in the project root.
7. The Python script must be fully self-contained and runnable without arguments.
8. Use today's actual date (from `date +%Y-%m-%d`) in all filenames and report headers.
9. Read actual source files before assigning ratings. Do not assess controls based on assumptions about the tech stack alone.
10. If this is a payment-processing system (evidence of PCI scope), flag PCI-DSS controls prominently in the gap analysis.
